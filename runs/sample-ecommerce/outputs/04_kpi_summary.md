# KPI 정의 및 기준값

> 분석 기준: MMP 수치 단일화 원칙 적용. 플랫폼 자체 수치는 비교 참조용으로만 병기.
> 데이터 범위: 2023-01-01 ~ 2023-12-30 / 집계 후 유효 행 수: 48,167행 (복합키 중복 1,833행 합산 처리)
> 단위 기준: spends 및 Conversion Sales 계열은 동일 단위(원 추정)로 취급. 절대값보다 채널 간 상대 비교에 집중.

---

## KPI 정의

| KPI | 계산식 | 단위 | 소스 | 목적 |
|---|---|---|---|---|
| **ROAS** (Return on Ad Spend) | `Conversion Sales_MMP ÷ spends` | 배수 (소수점 2자리) | MMP | 광고비 대비 매출 효율 측정. 채널·캠페인 간 핵심 비교 지표 |
| **CTR** (Click Through Rate) | `Clicks ÷ Impressions × 100` | % (소수점 2자리) | 플랫폼 (Clicks, Impressions) | 노출 대비 클릭 유도 효율. 소재·타깃 관심도 측정 |
| **CVR** (Conversion Rate) | `Conversions_MMP ÷ Clicks × 100` | % (소수점 2자리) | MMP (분자) / 플랫폼 (분모) | 클릭 후 전환 전환율. 랜딩 페이지·퍼널 효율 측정 |
| **CPA** (Cost Per Acquisition) | `spends ÷ Conversions_MMP` | 원/건 (정수) | MMP (분모) / 플랫폼 (분자) | 전환 1건당 획득 비용. 채널 효율의 절대값 기준 |
| **CPM** (Cost Per Mille) | `spends ÷ Impressions × 1,000` | 원/천회 (정수) | 플랫폼 | 노출 1,000회당 비용. 채널 도달 단가 비교 |
| **채널별 예산 비중** | `채널 spends ÷ 전체 spends × 100` | % (소수점 1자리) | 플랫폼 | 예산 배분 현황 파악 및 재배분 논거 수립 |
| **install→signup 전환율** | `signup_MMP ÷ install_MMP × 100` | % (소수점 2자리) | MMP | 앱 설치 후 회원가입 퍼널 전환 효율. 채널별 사용자 품질 측정 |

### KPI 정의 보충 원칙

- **ROAS**: `Conversion Sales_MMP`(MMP 기준 매출)만 사용. `Conversion Sales`(플랫폼 기준)는 사용하지 않음.
- **CVR**: 전환 분자는 반드시 `Conversions_MMP`. `Conversions`(플랫폼)는 사용 금지.
- **캠페인 목적별 CVR**: 캠페인 목적(install/signup/purchase)에 따라 전환 정의가 달라질 수 있으므로, 캠페인 세그먼트 분석 시 해당 전환 지표를 명시.
- **단위 혼용 금지**: ROAS는 배수, CTR/CVR/전환율은 %, CPA/CPM은 원/건(회). 동일 테이블에 혼용하지 않음.

## MMP 0값 처리 방침

### 방침: 0값 행 포함 — 공식 기준

**Conversions_MMP = 0인 행 7,373개 (전체 48,167행의 15.3%)를 집계에 포함한다.**

| 항목 | 값 |
|---|---|
| 해당 행 수 | 7,373행 |
| 전체 대비 비율 | 15.3% |
| 채널별 분포 | youtube 1,291 / naver_search 1,244 / google 1,232 / meta 1,216 / naver 1,203 / kakao 1,187 (고르게 분포) |
| 해당 행의 공통 특성 | Conversions_MMP = 0이면 Conversion Sales_MMP = 0, signup_MMP = 0도 동시에 해당 |

**포함 근거:**
- 0값 행은 광고가 실제 집행된 날짜의 기록이며, 해당 날 MMP 전환이 없었음을 의미한다.
- 제외할 경우 ROAS·CPA 수치가 인위적으로 좋아 보이는 선택 편향이 발생한다.
- 채널별로 고르게 분포(각 약 1,187~1,291행)하여 채널 간 비교에 왜곡이 없다.

**0값 포함 vs 제외 시 KPI 변화:**

| KPI | 0값 포함 (공식) | 0값 제외 (참조) | 차이 |
|---|---|---|---|
| ROAS (전체) | 10.98 | 12.89 | +1.91 (+17.4%) |
| CVR (%) | 3.25 | 3.81 | +0.56%p |
| CPA (원) | 12,779 | 10,889 | -1,890원 (-14.8%) |

> 0값 제외 버전은 분석 민감도 확인용으로만 참조. 의사결정·목표 설정에는 0값 포함 기준(10.98)을 사용.

## @analysis 에게 전달하는 주의사항

### 반드시 지켜야 할 계산 기준

1. **ROAS 계산**: `Conversion Sales_MMP ÷ spends`. `Conversion Sales`(플랫폼) 사용 시 10.4% 과대집계 발생 — 절대 사용 금지.
2. **CVR 계산**: `Conversions_MMP ÷ Clicks`. 분자는 MMP, 분모는 플랫폼(Clicks). 일관성 유지.
3. **CPA 계산**: `spends ÷ Conversions_MMP`. `Conversions`(플랫폼) 분모 사용 금지.
4. **0값 포함 원칙**: Conversions_MMP = 0인 7,373행을 포함하여 계산. 제외 버전은 민감도 확인용으로만 별도 표기.
5. **집계 순서**: 반드시 복합키(Channel·Date·Campaign·Ad Group·Ad Name) 기준 합산 집계(48,167행) 후 KPI 계산. 원본 50,000행 기준 계산 금지.
6. **install→signup 전환율**: `signup_MMP ÷ install_MMP`. 분모가 0인 행 제외하고 계산.

### 함정이 될 수 있는 수치

- **플랫폼 ROAS(12.12) vs MMP ROAS(10.98)**: 10.4% 차이. 채널별 성과 보고 시 항상 MMP 기준으로 명시. 플랫폼 수치를 실수로 사용하면 naver ROAS가 약 19.6으로 더 높게 나타남.
- **naver vs naver_search 개별/합산 혼용**: 개별 ROAS(17.92 / 17.72)와 합산 ROAS(17.82)는 유사하지만, 테이블마다 처리 기준을 명시하지 않으면 독자 혼동 유발.
- **CVR의 채널 간 차이가 작음(3.19~3.30%)**: 채널별 CVR 격차가 0.11%p에 불과해 절대값 비교보다 채널 내 시계열 추이 또는 캠페인×채널 교차 분석이 더 유의미할 수 있음.
- **install→signup 전환율의 채널 간 차이 미미(59.79~60.74%)**: 이 지표도 채널 간 차이가 약 1%p에 그침. 퍼널 분석 시 채널보다 캠페인 목적별 분리가 더 유효할 수 있음.
- **spends 단위 미확정**: 절대값(원 추정)보다 채널 간 상대 비교(ROAS·CPA·CPM) 에 집중. 단위 가정이 필요할 경우 "원 단위 추정" 을 명시.
- **retargeting CPA(13,170~13,400원)**: lookalike CPA(12,428~12,560원) 대비 7~8% 높음. ROAS 역전과 함께 retargeting 운영 비효율의 핵심 근거. 채널별 교차 분석 시 이 패턴이 전 채널에서 일관되는지 확인 필요.
- **MMP 이상치**: Conversions_MMP 최댓값 528, signup_MMP 최댓값 84, install_MMP 최댓값 148은 이상치로 분류(IQR 기준). 집계 수치에 포함하되, 고성과 날짜 vs 데이터 오류 여부를 세부 분석 시 주의.
