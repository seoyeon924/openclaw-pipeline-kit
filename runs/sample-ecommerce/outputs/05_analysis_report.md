# 분석 리포트

## 분석 전제
- **KPI 기준**: MMP 단일화 (Conversion Sales_MMP, Conversions_MMP, signup_MMP)
- **데이터 처리**: 복합키(Channel+Date+Campaign+Ad Group+Ad Name) 합산 집계 → 48,167행, MMP 0값 행 포함
- **ROAS** = Conversion Sales_MMP / spends
- **CVR** = Conversions_MMP / Clicks
- **CPA** = spends / Conversions_MMP

## 문제 2: retargeting 성과 역전

### 타깃 유형별 KPI

| 타깃 유형 | 지출(억원) | ROAS | CVR | CPA(원) |
|---------|-----------|------|-----|---------|
| lookalike | 569.0 | **11.22** | 3.27% | 12,493 |
| UA | 380.3 | 10.91 | 3.27% | 12,975 |
| retargeting | 192.1 | **10.44** | 3.27% | 13,286 |

> 통상적 기대(retargeting > UA > lookalike)와 반대. retargeting이 lookalike보다 ROAS 7.4% 낮음 (11.22 vs 10.44).

### 채널별 타깃 유형 ROAS 세분화

| 채널 | UA | lookalike | retargeting |
|------|-----|----------|------------|
| google | 7.66 | 8.23 | 7.60 |
| kakao | 7.85 | 8.23 | **7.56** |
| meta | 11.13 | 11.38 | 11.07 |
| naver | 18.08 | 18.19 | **16.80** |
| naver_search | 17.56 | 18.16 | **16.72** |
| youtube | 7.56 | 7.94 | 7.56 |

> retargeting 역전이 **모든 채널에서 동일하게 발생**. 채널 특이 문제가 아닌 리타게팅 운영 전략 자체의 문제. naver 계열에서 격차가 가장 큼 (UA 대비 -7.1~-7.4%).

### 월별 추이 (retargeting ROAS)
12개월 내내 retargeting이 lookalike보다 낮은 상태로 일관 — 광고 피로 누적 또는 리타게팅 모수 품질 문제일 가능성.

## 문제 3: CVR 분석

### 채널별 CVR

| 채널 | CVR |
|------|-----|
| google | 3.25% |
| kakao | 3.30% |
| meta | 3.27% |
| naver | 3.27% |
| naver_search | 3.19% |
| youtube | 3.19% |

> CVR이 3.19~3.30%로 전 채널 수렴. **CVR은 채널 성과 차이를 설명하지 못함** — ROAS 격차의 원인은 클릭당 전환가치(구매금액 차이)에 있음.

### 고노출-저CVR 구간
- 해당 행: 2,833개 (전체의 5.9%), 지출 약 50억원
- 채널 분포 균등 (naver_search 504행, youtube 488행, ... 고른 분포)
- 전략적 의미: CVR 개선보다 **전환 당 단가(ARPU) 최적화**가 더 핵심 레버

### 고성과 캠페인 특성 (CVR 상위 10%)
- 채널 분포 거의 균등 (kakao 845 ~ youtube 770)
- CVR 자체보다는 **캠페인 타깃 정밀도와 랜딩 최적화** 차이로 추정

- **반드시 시각화할 수치**: 채널별 ROAS 바차트 (예산비중과 함께 표시), 예산 재배분 시나리오 비교

- MMP 기준 수치만 사용, 플랫폼 수치(ROAS 12.12) 혼용 금지
